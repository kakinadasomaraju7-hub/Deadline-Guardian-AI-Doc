# 📊 Deadline Guardian AI

An AI-powered Task Management & Analytics Dashboard that helps users track tasks, monitor productivity, and generate smart study/work schedules in real time.

---

## 🚀 Features

- ➕ Add / Update / Delete tasks
- ✅ Mark tasks as completed
- 📊 Live analytics dashboard
- 📈 Task completion tracking
- ⏱ Real-time activity monitoring
- 🤖 AI-based schedule planner (optional feature)
- ☁️ Firebase backend support (Firestore integration)

---

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript
- Firebase (Firestore, Authentication)
- Chart.js (for analytics graphs)
- Node.js (optional backend support)
